 HireReady AI

**AI-powered Resume Builder & Interview Preparation Platform**

HireReady AI helps students and job seekers create ATS-optimized
resumes, prepare for interviews, and generate professional PDF resumes
using AI and multiple LaTeX templates.

## ✨ Features

-   📄 AI-powered ATS Resume Builder
-   🎨 Multiple Resume Templates (ATS Classic, Modern, Professional)
-   🤖 AI Resume Enhancement using Gemini
-   📑 LaTeX-based PDF Resume Generation
-   🎯 AI Interview Preparation & Analysis
-   📊 Resume Analysis & ATS Optimization
-   🔐 Secure JWT Authentication

## 🛠 Tech Stack

-   Node.js

-   MongoDB

### AI

-   Google Gemini API

### Resume Generation

-   LaTeX

## 🚀 Installation

``` bash
git clone https://github.com/your-username/HireReady-AI.git
cd HireReady-AI
```

### Backend

``` bash
cd backend
npm install
npm run dev
```

### Frontend

``` bash
cd frontend
npm install
npm run dev
```
